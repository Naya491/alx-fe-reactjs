# ALX React App

This is a simple React project bootstrapped with [Vite](https://vitejs.dev/).

## 🚀 Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/alx-react-app.git
   cd alx-react-app
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start development server
   ```bash
   npm run dev
   ```
